# TP RASA
## Installation
### Base de donnée
Pour créer la base de données, veuillez lancer le script [generatedatabase.py](./generatedatabase.py) à la racine du projet
```bash
$ python3 ./generatedatabase.py
```

### Schell
Pour tester le dernier model lancer le shell
```bash
$ rasa shell
```
